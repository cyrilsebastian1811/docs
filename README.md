## Commands

* `mkdocs new [dir-name]` - Create a new project.
* `mkdocs serve -a 0.0.0.0:8000 --config-file mkdocs.yml` - Start the live-reloading docs server.
* `mkdocs build --site-dir site` - Build the documentation site.
* `mkdocs -h` - Print help message and exit.
